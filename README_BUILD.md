#dev-drones

You can run test and application from root dev-drones using:
```
./mvnw test
```
For integration test and run application you need launch Docker Desktop because 
application use PostgresSQL and LocalStack Docker containers
```
./mvnw clean spring-boot:run
```

Documentation is generated by Swagger 3.0. You can find descriptions dto and controller endpoints here:
```
http://localhost:8080/swagger-ui/index.html#/
```
